3. Token được lưu trong trình duyệt
4. Quên token phải xóa localStorage

Quản lý GitHub

Token được lưu an toàn trong localStorage

Hướng dẫn

  1. Vào GitHub Settings → Developer settings → Personal access tokens
  2. Chọn "Generate new token (classic)"
  3. Tick chọn quyền: repo (Full control of private repositories)
  4. Copy token và paste vào ô trên
